## BMX Race System

BMX sacensību pārvaldības mājaslapa, izstrādāta ar Flask un SQLite.

## Funkcijas
- Braucēju pievienošana
- Braucēju rediģēšana
- Sacensību grafiks
- Pieteikšanās sacensībām
- Rezultātu apskate

## Izmantotās tehnoloģijas
- Python
- Flask
- SQLite
- HTML
- CSS
- GitHub
- ChatGPT
